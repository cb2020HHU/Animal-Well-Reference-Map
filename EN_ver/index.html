Animal Well Reference Map

This is the map of the game "Animal Well". I think it's not convenient to use the existing online maps, so I made this one. The information (positions, tips, etc.) is from the website techraptor.net, you can check their articles for more details.

This work is for convenience, if you haven't cleared the game yet, try to solve the puzzles by yourself.

There are some rooms on the edge of the map that can be connected to the other side, and there are letters at the corresponding positions.

On many other pages you can click the item image and jump to the position on the map, whereas on this page you can change the layer in the following map to show different items: use the Top-Right corner button to change the layers, use the mouse scroll wheel to zoom in and out, press and hold the map to drag it.

I may update this work in the future. --by C.B.
